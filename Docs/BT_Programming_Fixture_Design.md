# MedFlow POSEIDON (滴護寶) 10路並行燒錄治具設計說明書 (2x5 矩陣壓合版)

本文件為 **MedFlow 滴護寶 (POSEIDON) 藍牙模組 PCBA** 之工廠量產用 **10路並行燒錄治具** 設計規格。為了大幅提升工廠端燒錄效率，本治具採用 **10通道並行燒錄架構**（2x5 矩陣排列），內部整合 **USB Hub 擴展晶片** 與 **10組獨立的 USB-TTL 串口晶片**，實現只需一條 USB 線連接電腦即可同時燒錄 10 片藍牙模組。

---

## 1. 燒錄治具外觀與機械結構 (10通道版)

治具採用平價電木底座與透明壓克力手動鉸鏈蓋板結構，作業員可一次放入 10 片進行燒錄與測試。

### 1.1 機械與定位設計細節
* **10通道 2x5 對位凹槽 (PCBA Alignment Grooves)**：
  * 電木底座表面精密銑出 **10 個呈 2x5 矩陣排列的方形凹槽**，每個凹槽尺寸完美吻合 25mm x 25mm 藍牙模組 PCBA 的邊緣。
  * 凹槽底部設有防呆下陷，使 PCBA 只能以單一方向置入，避免方向反置。
* **J1 彈片座模擬對接（金手指舌片）**：
  * 每個凹槽的左側水平突出一個厚度與觸控貼片相同的硬質 PCB 金手指舌片（模擬 FPC 插頭）。
  * 當作業員將模組 PCBA 放入凹槽並向左側推進時，PCBA 板載的 **J1 黑色插槽** 會順勢滑入此舌片中，完成供電與 KEY1 驗證端的連接。
* **J2 探針壓合（單蓋 40 根 Pogo Pins）**：
  * 治具上蓋（透明壓克力板）配置有 **10 組、共 40 根黃金彈簧探針 (Pogo Pins)**。
  * 當作業員蓋下上蓋並鎖定快速夾鉗時，40 根探針剛好垂直壓入並抵緊 10 片 PCBA 上的 **J2 UART 孔位**，不需要人工逐片用手頂接。
* **獨立 LED 狀態燈**：
  * 每個凹槽旁配置一顆雙色（紅/綠）LED 指示燈，分別接在對應通道的 UART 狀態指示或自檢輸出。綠燈代表燒錄完成/正常，紅燈代表燒錄失敗/出錯。

## 2. 10路並行電路與電腦連線架構

為實現「一線連電腦，多路同時燒」，治具控制板內部整合了高速 USB 集集線器 (USB Hub) 晶片與 10 組獨立的串口晶片：

```
                              [ 電腦 PC ]
                                  │
                                  ▼ (一條 USB 傳輸線)
                         ┌─────────────────┐
                         │   燒錄治具主板  │
                         │ (板載 USB Hub)  │
                         └────────┬────────┘
                                  │
      ┌───────┬───────┬───────┼───────┬───────┬───────┬───────┬───────┐
      ▼ (CH1) ▼ (CH2) ▼ (CH3) ▼ (CH4) ▼ (CH5) ▼ (CH6) ▼ (CH7) ▼ (CH8) ▼ (CH9) ▼ (CH10)
    ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐
    │USB- │ │USB- │ │USB- │ │USB- │ │USB- │ │USB- │ │USB- │ │USB- │ │USB- │ │USB- │
    │ TTL │ │ TTL │ │ TTL │ │ TTL │ │ TTL │ │ TTL │ │ TTL │ │ TTL │ │ TTL │ │ TTL │
    └──┬──┘ └──┬──┘ └──┬──┘ └──┬──┘ └──┬──┘ └──┬──┘ └──┬──┘ └──┬──┘ └──┬──┘ └──┬──┘
       │      │      │      │      │      │      │      │      │      │
     [CH1]  [CH2]  [CH3]  [CH4]  [CH5]  [CH6]  [CH7]  [CH8]  [CH9]  [CH10] 探針端子(J2)
```

### 2.1 電氣規格與晶片配置
* **USB Hub 擴展晶片**：採用兩顆串聯的 **FE1.1S**（或一顆 **GL852G** 7口 Hub + 一顆 FE1.1S）集線器晶片，將治具的 USB Type-C 輸入擴展為 10 個 USB 下行埠。
* **USB-TTL 晶片**：板載 10 顆 **CH340N** 晶片，分別連接至 Hub 的 10 個下行埠。當治具插上電腦後，電腦端會**同時識別出 10 個虛擬串口 (COM Ports)**。
* **並行燒錄軟體邏輯**：
  * 電腦端燒錄程式（Actions 燒錄腳本）會同時開啟 10 個執行緒/行程 (Parallel Flashing Threads)，分別向 COM1 ~ COM10 同步下發韌體資料。
  * 10 個通道互不干擾，即使其中一個通道發生接觸不良，也不會影響其他 9 個通道的寫入。
* **防倒灌供電與短路防護**：
  * 每個通道的 VCC 供電腳均設有獨立的 **Diodes 防倒灌二極體** 與 **100mA 自恢復保險絲 (PPTC)**，防止個別板子損壞時連帶造成集線器或電腦 USB 接口當機。

## 3. 測試與燒錄接口定義 (各通道獨立)

每個測試通道的 J2 孔位壓接引腳與 J1 模擬舌片定義如下：

### J2 探針連接介面 (UART 燒錄通道)
* **Pin 1 (VDD)**：治具供電 VCC 3.3V。
* **Pin 2 (TX)**：UART0_TX -> 連接該通道獨立 CH340N 的 **RXD**。
* **Pin 3 (GND)**：系統接地 PGND。
* **Pin 4 (RX)**：UART0_RX -> 連接該通道獨立 CH340N 的 **TXD**。

### J1 模擬舌片連接介面 (自檢自測通道)
* **+VBAT1 與 +VBAT2 短路**：接通供電通路。
* **KEY1 接收**：連接至治具狀態自檢輸入。

## 4. 工廠燒錄作業流程

1. **裝載 PCBA**：作業員將 10 片裸板 PCBA 依序放進 10 個凹槽內，並向左滑動推入使 J1 插進模擬舌片中。
2. **壓合蓋蓋**：下壓壓克力上蓋，快速夾扣鎖定，**40 根 Pogo Pins 壓入 10 片板子的 J2 孔內**。
3. **並行燒錄**：電腦端燒錄腳本偵測到 10 個 COM 埠，並行呼叫 Action Burner 下發韌體。
4. **狀態判定**：
   * 綠燈常亮：PASS
   * 紅燈常亮：FAIL
5. **卸載出料**：打開蓋板，挑出紅燈的不良品，綠燈的良品收納至出貨托盤。
