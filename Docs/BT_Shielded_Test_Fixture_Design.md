# MedFlow POSEIDON (滴護寶) 50路射頻屏蔽測試治具設計說明書 (5x10 獨立屏蔽腔版)

本文件為 **MedFlow 滴護寶 (POSEIDON)** 藍牙貼片之工廠量產用 **50路藍牙射頻屏蔽測試治具** 設計規格。針對工廠環境中嚴重的 Wi-Fi、藍牙及其他射頻訊號干擾，以及 50 片貼片同時廣播造成的同頻干擾問題，本治具採用**主體屏蔽箱**、**CNC 獨立金屬屏蔽腔（法拉第籠）**、**近場射頻耦合天線**與**射頻多路開關矩陣**，實現精準且互不干擾的批量藍牙 RF 性能量測。

---

## 1. 測試治具外觀與機械結構 (射頻屏蔽版)

治具外觀採用重型氣動或手動壓接屏蔽箱設計，作業員可一次放入 50 片進行自檢與射頻性能量測。

### 1.1 機械與射頻隔離設計細節
* **外層氣動/手動屏蔽箱 (RF Shielding Box)**：
  * 最外層為高強度鑄鋁/防銹鋼板製成的屏蔽箱，邊緣配置有**導電金屬彈片與導電橡膠密封條**。
  * 關閉屏蔽蓋後，可阻絕外部廠房環境中 2.4GHz 頻段達 **>70dB 的射頻衰減**，確保外部干擾信號無法進入。
* **單槽 CNC 鋁合金隔離腔 (Individual Aluminum Cavities)**：
  * 下電木板上方貼合一塊經由 CNC 加工的整塊 **鋁合金隔離腔體板 (5x10 矩陣，共 50 腔)**。
  * 每個腔體深度剛好容納一片滴護寶貼片，腔體四周邊緣黏貼有**導電泡棉 (Conductive Gaskets)**。
  * 當壓下上壓蓋時，上蓋對應的 50 個鋁合金防護罩會與下底座的鋁合金腔體完全貼合，使 50 片藍牙貼片分別被密封在 **50 個獨立的法拉第籠 (Faraday Cage)** 中。
  * 腔體間的射頻隔離度 (RF Isolation) 可達 **>40dB**，徹底杜絕相鄰貼片間的同頻干擾。
* **近場耦合天線 (Near-field Coupling Antenna)**：
  * 在每個屏蔽腔體的上蓋正對貼片藍牙天線的位置，嵌入一個**陶瓷近場耦合天線 (RF Coupler)**。
  * 天線與貼片維持固定 10mm 的微小距離，確保射頻信號能穩定、小範圍地定向耦合，避免輻射到空間中。

---

## 2. 射頻信號與多路切換電路 (RF Switch Matrix)

為了解析 50 個通道的藍牙信號，治具採用**射頻開關矩陣 (RF Switch Matrix)**，將 50 路天線信號分時切換引入測試接收端。

### 2.1 射頻切換與檢驗電路設計
1. **射頻多路開關 (RF MUX)**：板載多顆低損耗 2.4GHz 射頻開關 IC（如 **PE4259** 或 **HMC253** 8選1射頻開關），級聯組成 50選1 的射頻開關矩陣。
2. **雙屏蔽同軸線路**：每個耦合天線至開關主板的連接線，皆採用帶有 IPEX 接頭的雙層屏蔽微細同軸線（如 **RG178**），防止線路上產生信號洩漏與交擾。
3. **測試接收器 (Tester Receiver)**：
  * 採用治具專用的 **Poseidon Gateway** 控制電路板作為標準測試接收端。
  * 主控 MCU 透過 GPIO 控制射頻矩陣依次導通第 1 至第 50 通道。當導通某個通道時，測試接收器專門接收該通道的 BLE 廣播封包。

## 3. 射頻性能判定指標 (RF Test Parameters)

在隔離外界干擾的狀況下，治具可對每個通道進行定量的射頻性能判定：
* **接收強度比對 (RSSI Calibration)**：在固定的近場天線耦合距離與 -10dB 衰減下，正常貼片的 RSSI 應穩定在 **$-45\text{ dBm} \pm 3\text{ dB}$** 區間內。若 RSSI 衰減至 $-60\text{ dBm}$ 以下，代表天線阻抗不匹配、功率放大電路瑕疵或焊接不良 (FAIL)。
* **廣播丟包率 (Packet Delivery Rate, PDR)**：貼片廣播間隔為 5秒。治具切換至該通道後會監聽 20秒。在 20秒內必須成功接收到至少 3 個完整的廣播封包，否則視為丟包率過高或天線輻射能力差 (FAIL)。
* **頻率偏移 (Frequency Offset)**：測試端接收解析廣播載波，檢測載波中心頻率偏離值是否在 **$\pm 20\text{ ppm}$** 以內。高於此值代表晶振 (12MHz) 頻偏超標，會導致後續連線不穩定。

## 4. 電氣、觸控與射頻同步測試流程

治具將「射頻測試」與原先的「電壓、觸控自檢」結合在同一套下壓過程中：
1. **下料對位**：作業員將 50 片貼片放入電木凹槽，推入使 **J1 插槽接上彈片座**。
2. **壓合鎖定**：關閉氣動/手動屏蔽箱上蓋。此時：
   * 外部 RF 被絕緣箱完全屏蔽。
   * 50 個單槽鋁合金隔離腔完全閉合，貼片被彼此隔離。
   * 治具上的橡膠壓塊壓緊貼片，接通電源。
3. **電壓與漏電檢測**：順序量測 50 通道的電池 OCV、CCV 與待機靜態電流（如前述電晶體控制）。
4. **自動化模擬觸控測試**：控制 MOSFET 導通電容模擬極板，讀取 `KEY1` 是否響應。
5. **射頻掃描與 RSSI 測試**：主控 MCU 切換射頻開關矩陣至通道 1，監聽 15-20 秒，量測其 RSSI、頻偏與 PDR。順序切換至通道 2... 直至通道 50。
6. **面板亮燈與報告輸出**：治具蓋板掀開後，主板上測試失敗的槽旁亮**紅燈**，測試通過的亮**綠燈**。
