# MedFlow POSEIDON (滴護寶) 50通道量產測試治具設計說明書 (5x10 彈片座槽版)

本文件為 **MedFlow 滴護寶 (POSEIDON)** 觸控偵測貼片的工廠量產用測試治具設計規格。配合工廠對對位便利性與低製作成本的要求，本治具設計一次測試 **50 片 (5x10 矩陣排列)**，機械定位結構採用**電木防呆對位凹槽**，接腳接觸端子採用與主板同款的 **4-Pin 黑色金手指插槽 (彈片座)**，顯示界面使用**單槽獨立板載 LED 狀態指示燈**。

---

## 1. 測試治具外觀與機械結構 (5x10 彈片座版)

治具採用平價耐用材料與高效率防呆定位設計，作業員可一次放入 50 片測試物。

### 1.1 機械與定位設計細節
* **單槽定位防呆凹槽 (Alignment Grooves)**：
  * 治具的下電木板板面加工有 **50 個呈 5x10 矩陣排列的對位凹槽**。
  * 凹槽外型與貼片的藍牙本體及 FPC 尾翼完全吻合，作業員只需將貼片輕輕放入對位凹槽中即可實現完美對齊，避免偏斜導致金手指接觸不良。
* **接觸端子（黑色彈片座）**：
  * 每個凹槽的尾端配置一個與 POSEIDON 主板同款的 **4-Pin 黑色直角/垂直金手指插槽 (即 J1 彈片座)**。
  * 貼片放入凹槽時，其尾端金手指順勢插進該黑色彈片座中，利用插槽內部的黃金彈片夾緊，提供最穩定、最貼合實際產品運作狀態的電氣接觸。
* **下壓蓋板機構**：
  * 上方採用一片帶有防靜電橡膠壓塊的透明壓克力板，透過後紐手動下壓。壓下時，橡膠壓塊會均勻施壓在 50 片貼片的金手指插接處，確保插接牢固不鬆脫。
* **一槽一燈 LED 顯示**：
  * 每個定位凹槽旁直接配置一組雙色（紅/綠）LED 指示燈。綠燈亮代表 PASS，紅燈亮代表 FAIL，哪一槽有問題一目了然，便於作業員直接挑出不良品。

## 2. 測試物 (UUT) 接口與迴路設計

觸控貼片尾端金手指共 4 Pin，其物理定義與治具的迴路設計如下：

| 金手指 Pin 名稱 | 功能定義 | 貼片內部迴路 | 治具端對接與測試原理 |
| :--- | :--- | :--- | :--- |
| **+VBAT1** | 電池正極輸出 | 直接連接 CR2032 鈕扣電池正極 | 1. 治具端高阻抗 ADC 測量其開路電壓 (OCV)。<br>2. 經由治具內部開關，將其與 +VBAT2 短路以供電給觸控 IC，測量負載電壓 (CCV)。 |
| **+VBAT2** | 觸控 IC 電源輸入 | 連接觸控 IC (BS211C) VDD 電源腳 | 作為節能設計，貼片平時此腳懸空（電池與 IC 隔離）。<br>測試時治具透過彈片座接通此迴路使貼片開機。 |
| **PGND** | 系統接地 (GND) | 連接電池負極與觸控 IC GND | 治具系統共同接地端。 |
| **KEY1 / TPQ0** | 觸控 GPO 狀態輸出 | 觸控 IC 輸出腳 (1: 有水/觸碰, 0: 無水/未觸) | 治具連接至 GPIO 輸入腳，讀取狀態高低電平。 |

## 3. 電容式觸控功能「自動化模擬」測試原理

在工廠批量測試中，無法使用人工手指去觸碰 50 片貼片來驗證觸控功能。因此治具在貼片下方的定位凹槽中嵌入了**電容負載模擬電極**：

```
[ 測試壓板 ]  ======== 壓下壓板壓緊 ========
[ 貼片正面 ]  [ 藍牙貼片本體 (CR2032 & IC) ]
[ 貼片背面 ]  [ 銅箔感應區 (白色膠片面) ]
----------------- (治具定位凹槽底面，薄絕緣膜) -----------------
[ 治具底座 ]  [ 嵌入式銅質電極板 ]
                     │
                     ▼ (導線)
                 ┌───┴───┐
                 │D     S│
              [N-MOSFET] ───> [治具 GND]
                 │G (控制端)
                 └───┬───┘
                     ▼ 
         [治具 MCU 控制訊號 (Touch_Sim)]
```

### 3.1 測試原理與步驟
1. **絕緣設計**：治具定位凹槽底面貼有極薄的絕緣 Kapton 膠帶（厚度約 0.05mm - 0.1mm），當貼片放入時，其背面的銅箔與治具的銅質電極板形成電容耦合，但無直流導通。
2. **模擬「未觸碰」狀態**：治具控制端輸出 Low 給 N-MOSFET 閘極（Gate），使 MOSFET 截止（S/D 開路）。此時電極板處於浮接狀態，對貼片的對地電容量極小，觸控 IC 應輸出 **Low**（未觸碰）。
3. **模擬「觸碰」狀態**：治具控制端輸出 High 給 N-MOSFET 閘極，使 MOSFET 飽和導通（S/D 短路，電極板接地）。此時，電極板引進了巨大的對地寄生電容，等同於人體手指觸碰感應區，觸控 IC 偵測到電容變化後，應輸出 **High**（觸碰）。

## 4. 50 通道多路複用 (Multiplexing) 硬體架構

配合通道數下修至 50 路，控制板的硬體線路得以簡化，元件成本進一步降低：

```
              ┌───────────────────────────────────────────────┐
              │             控制板主 MCU (ESP32)              │
              └──────┬─────────────────┬───────────────┬──────┘
                     │SPI              │I2C            │SPI
                     ▼                 ▼               ▼
           ┌──────────────────┐ ┌─────────────┐ ┌─────────────┐
           │   13顆 74HC595   │ │ 4顆 MCP23017 │ │4顆 CD74HC4067│
           │  (串列轉並列晶片) │ │ (I/O 擴展晶片)│ │ (類比多路開關)│
           └─────────┬────────┘ └──────┬──────┘ └──────┬──────┘
                     │100路控制線      │50路           │50路
                     ▼                 ▼               ▼
           ┌──────────────────┐ ┌─────────────┐ ┌─────────────┐
           │ 觸碰(50路)及     │ │ 讀取貼片    │ │ 測量貼片    │
           │ 電源開關(50路)   │ │ KEY1 (GPO)  │ │ +VBAT1 電壓 │
           └──────────────────┘ └─────────────┘ └─────────────┘
```

### 4.1 關鍵元件清單與通道擴展
1. **模擬觸碰與電源迴路控制**：
   * 共有 50 路模擬觸碰 MOSFET 閘極，以及 50 路用來切斷/開啟測試槽供電的 P-MOSFET 控制線，共計 100 條數位控制輸出。
   * 使用 13 顆 8 位元移位暫存器 **74HC595** 級聯控制，可完全滿足 100 條控制線路需求。
2. **觸控信號讀取 (50路)**：
   * 讀取各通道 `KEY1` 的電平訊號。
   * 使用 4 顆 16 位元 I/O 擴展晶片 **MCP23017**（可提供 64 個 GPIO 輸入，多出 14 腳作預留）。
3. **電池電壓量測 (50路)**：
   * 測量各貼片 `+VBAT1` 的電池電壓。
   * 使用 4 顆 16 通道類比多路選擇器 **CD74HC4067**（可提供 64 個類比通道，多出 14 通道作預留），將信號導向主控 MCU 的 ADC 輸入腳。

## 5. 自動化測試流程與判定指標

治具啟動後，執行以下快速掃描測試流程：
1. **量測開路電壓 (OCV)**：檢測 `+VBAT1` 與 `PGND` 開路電壓。低於 2.8V 判定電量不足，亮該槽**紅燈**。
2. **待機電流測試**：接通該槽 `+VBAT1` 與 `+VBAT2` 供電，測量電流。若高於 5uA，判定漏電/短路，亮該槽**紅燈**。
3. **未觸碰測試**：保持模擬觸控 MOSFET 斷開，讀取 `KEY1`，若為高電平，判定誤動作，亮該槽**紅燈**。
4. **模擬觸碰測試**：閉合模擬觸控 MOSFET 使電極板接地，讀取 `KEY1`，若未轉為高電平，判定無反應，亮該槽**紅燈**。
5. **通道判定**：全部合格則該槽亮**綠燈**，表示 PASS。

### 5.1 測試判定標準
* **電池電壓合格線**：$V_{\text{BAT}} \ge 2.8\text{V}$
* **待機靜態電流**：$\le 3\mu\text{A}$
* **觸控反應時間**：$\le 150\text{ms}$
