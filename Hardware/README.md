# MedFlow 硬體設計說明 (Hardware Design Notes)

本目錄主要儲存 **滴護寶 (Smart IV Drip Monitor)** 的硬體開發相關檔案，包括 KiCad 電路圖、電路板配置 (PCB Layout)、物料清單 (BOM) 與測試紀錄。

---

## 1. 硬體設計架構

滴護寶硬體系統由以下子系統構成：

```
+-------------------------------------------------------------------+
|                           滴護寶 PCBA                             |
|                                                                   |
|  +------------------+     +------------------+    +------------+  |
|  |   紅外線滴速感測   | --> |                  |    |  電源管理  |  |
|  |   (GPIO/比較器)  |     |                  |    | (鋰電池充電|  |
|  +------------------+     |                  |    | + 低靜態LDO|  |
|                           |     主控 MCU     |    | + 電量計)  |  |
|  +------------------+     |   (Wi-Fi/BLE)    |    +------------+  |
|  |   高精度重量感測   | --> |                  |           |      |
|  |   (24-bit ADC)   |     |                  | <---------+      |
|  +------------------+     +------------------+                    |
|                                |        |                         |
|      +-------------------------+        +---------+               |
|      |                                            |               |
|  +--------+                                  +---------+          |
|  | 本地UI | (RGB LED / 蜂鳴器 / 按鍵)        | 除錯UART|          |
|  +--------+                                  +---------+          |
+-------------------------------------------------------------------+
```

## 2. 子電路設計細節

### A. 電源管理系統 (Power Management)
* **輸入電源**：5V USB-C 充電輸入。
* **充電管理**：使用線性鋰電池充電晶片（如 TP4056 或具有電源路徑管理的晶片），最大充電電流設定為 500mA，配合過充/過放保護電路。
* **系統供電**：低靜態電流 (Low-Iq) 的 3.3V LDO（如 XC6206 或 AP2112），確保進入休眠模式 (Deep Sleep) 時的耗電量控制在數微安培 ($\mu\text{A}$) 等級。
* **電量檢測**：加入分壓電阻電路或專用電量計 (Fuel Gauge, I2C 介面) 來精確監測單節鋰電池的電壓與電量狀態。

### B. 感測器介面 (Sensors Interface)
* **滴速感測 (Infrared Drip Counter)**：
  * 使用紅外線發射與接收對射管夾在點滴滴斗兩側。
  * 接收端輸出訊號送至主控 MCU 具有外部中斷 (GPIO Interrupt) 功能的引腳，以便在每一滴點滴經過時喚醒計數器。
* **重量感測 (Load Cell Weight Sensor)**：
  * 對應需要高精度剩餘容量評估的需求，設計微型點滴秤重感測器。
  * 採用阻抗式半橋或全橋稱重傳感器，並透過高精度 24-bit ADC（如 HX711 或 TI ADS1232）進行訊號放大與數位化，透過兩線式序列介面 (I2C 或類似協議) 與 MCU 通訊。

### C. 狀態指示與本地交互 (UI & Alarm)
* **RGB LED 狀態燈**：用於指示工作狀態（綠燈常亮表示正常，黃燈閃爍表示流速變慢，紅燈閃爍加蜂鳴器表示警告）。
* **蜂鳴器電路**：採用無源/有源電磁式蜂鳴器，作為本地緊急聲光警報。
* **功能按鍵**：包括一個復位鍵 (Reset) 與一個功能設定按鍵 (如網路配對、靜音警報等)。

## 3. 目錄檔案說明 (建議)

* `/PCBA`：KiCad 專案檔（`*.kicad_pro`, `*.kicad_sch`, `*.kicad_pcb`）。
* `/BOM`：物料清單電子表格與詢價單。
* `/Gerber`：送交 PCB 廠打樣的 Gerber 檔與鋼網資料。
* `/Datasheets`：關鍵元件的手冊與設計手冊。
